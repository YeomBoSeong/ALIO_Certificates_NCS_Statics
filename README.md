# ALIO 공기업 채용공고 수집기

ALIO 공개 API(`opendata.alio.go.kr`)를 이용해 공기업 채용공고를 수집하고,  
공고별 **요구 자격증**과 **NCS 과목**을 추출해 통계를 Excel 파일로 저장하는 프로그램입니다.
공기업 취직을 준비를 시작했을 때, 기업들이 어떤 자격증들에 가산점을 주는지, 어떤 NCS 과목을 준비해야하는지 

---

## 요구사항

- **Python 3.10 이상** ([python.org](https://www.python.org) 에서 설치)
- Windows 10 / 11

---

## 설치 및 실행

코드를 다운로드 후, 폴더에서 setup.bat을 실행한 후 (의존성 다운로드로, 다운로드 후 최초 1회만 실행), run.bat을 실행하면 됩니다.

---

## 사용 방법

### 설정 탭

#### 검색 키워드

| 탭 | 설명 |
|---|---|
| ALIO 직종 | ALIO 사이트의 공식 직종 분류 (전산직, 행정직, 연구직 등 25개) |
| NCS 분류 | NCS 24개 대분류 (정보통신, 경영·회계·사무 등) |
| 직접 입력 | 쉼표로 구분하여 직접 입력 (OR 검색)|

- 복수 선택 시 키워드 간에는 **OR** 검색입니다.  
  예) 전산직 + 행정직 선택 → 전산직 **또는** 행정직 공고 수집
- NCS 분류는 `·`(가운뎃점)을 기준으로 분리하여 각각 검색합니다.  
  예) `전기·전자` 선택 → `전기`, `전자` 두 키워드로 각각 검색

#### 포함 / 제외 단어 (직접 입력 탭 하단)

API에서 받아온 공고를 **기관명 + 공고 제목** 기준으로 추가 필터링합니다.  
API 자체는 단순 키워드 검색만 지원하므로, AND 조합과 제외는 여기서만 가능합니다.

| 문법 | 의미 | 예시 |
|---|---|---|
| `A \| B` | A 또는 B | `병원 \| 의료원` |
| `A & B` | A 그리고 B | `정보 & 보안` |
| `A \| B & C` | A 또는 (B AND C) | `전산 \| 정보 & 보안` |
| 비워두면 | 전체 허용 | |

#### 수집 옵션

- **키워드당 최대**: 키워드 하나당 수집할 최대 공고 수
- **첨부파일 다운로드**: 체크 시 HWP·PDF 파일을 다운로드해 NCS·자격증 추출 정확도가 높아집니다. 시간이 더 걸립니다.

#### 저장 위치

- Excel 결과 파일은 지정한 폴더에 저장됩니다.
- 첨부파일은 지정한 폴더 안의 `Downloads/` 폴더에 저장됩니다.
- 기본값: 프로그램 폴더(`RealProgram/`)

---

### 수집 진행 탭

수집 시작 후 자동으로 전환됩니다.

| 진행바 | 설명 |
|---|---|
| 위 | 공고 수집 및 첨부파일 다운로드 진행률 |
| 아래 | NCS·자격증 분석 진행률 |

로그에는 키워드별 수집 결과가 표시됩니다:

```
'전산직' 수집: 25개,  필터 제외: 64개,  전체 누적: 25개
'정보통신' 수집: 100개,  필터 제외: 0개,  키워드 중복: 5개,  전체 누적: 120개
```

- **필터 제외**: 포함/제외 단어 조건에 걸려 제외된 공고 수
- **키워드 중복**: 앞선 키워드에서 이미 수집된 공고 (중복 수집 방지)
- **전체 누적**: 현재까지 수집된 고유 공고 수

## 결과 파일 (Excel)

`분석결과_날짜시간.xlsx` 파일이 저장 위치에 생성됩니다.

| 시트 | 내용 |
|---|---|
| 공고별 | 기업명 · 공고제목 · 요구 자격증 · NCS 과목 |
| 자격증_통계 | 자격증명 · 공고 수 · 기업 수 · 해당 기업 (빈도 높은 순) |
| NCS_통계 | NCS 과목 · 공고 수 · 기업 수 · 해당 기업 |
| NCS_직기능 | NCS 직업기초능력 10종 · 공고 수 · 기업 수 · 해당 기업 |

### 자격증 추출 방식

1. API 응답의 자격요건 텍스트(`aplyQlfcCn`)에서 추출
2. 다운로드한 HWP·PDF 파일 파싱으로 보완

### NCS 추출 방식

1. API 응답의 NCS 코드명(`ncsCdNmLst`)에서 추출
2. HWP 직무기술서의 `<NCS 분류체계>` 섹션 파싱으로 보완
